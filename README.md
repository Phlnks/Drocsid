# Drocsid

This is a real-time chat application built with React, TypeScript, and a Node.js backend. It's designed to be a fast and reliable communication platform, with a focus on simplicity and ease of use.

## Features

*   Real-time messaging
*   Voice channels
*   User presence
*   Roles and permissions
*   File uploads
*   GIFs and emojis
*   Link previews
*   Screen sharing

## Getting Started

1.  Install the dependencies:

    ```bash
    npm install
    ```

2.  Start the development server:

    ```bash
    npm run dev
    ```

3.  Open your browser to `http://localhost:3000` to see the application.
